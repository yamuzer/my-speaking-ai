# API 사용량 관리 페이지 - 인터페이스 기획안

## 📋 기본 개요
API 사용량 관리 페이지는 사용자가 총 사용량과 비용을 한눈에 파악하고, 각 대화별 상세 사용량을 추적할 수 있는 페이지입니다.

---

## 🎯 주요 요구사항
- ✅ 총 사용량 및 비용 표시
- ✅ 기간별 필터링 (7일, 30일, 90일)
- ✅ 각 대화별 사용량 상세 조회
- ✅ 사용 시간 표시
- ✅ 정렬 및 검색 기능

---

## 🏗️ 페이지 레이아웃 구조

### 1️⃣ 헤더 섹션
```
┌─────────────────────────────────────┐
│ Usage                               │
│ API 사용량 관리         [추정치]     │
│ OpenAI API 사용량과 예상 비용을...   │
└─────────────────────────────────────┘
```

### 2️⃣ 제어 영역 (기간 선택 + 필터)
```
┌─────────────────────────────────────┐
│ 기간: [최근 30일▼]  [새로고침 버튼]  │
│ 정렬: [비용 높은순▼]  검색: [입력칸]  │
└─────────────────────────────────────┘
```

### 3️⃣ 요약 카드 영역 (총량 표시)

#### 3-1. 핵심 지표 카드 (3개 열 레이아웃)
```
┌──────────────┬──────────────┬──────────────┐
│  $$42.50     │  156.7분     │  1,234개     │
│  예상 비용   │  음성 대화   │  메시지      │
└──────────────┴──────────────┴──────────────┘
```

#### 3-2. 추가 지표 카드 (2개 열 레이아웃)
```
┌──────────────────────┬──────────────────────┐
│  84개                │  34개                 │
│  총 대화 세션        │  평균 세션 시간(분)   │
└──────────────────────┴──────────────────────┘
```

#### 3-3. 비용 분석 카드
```
┌─────────────────────────────────────────┐
│ 대화 비용 추정                           │
│ 음성 입력/출력 토큰 기준...             │
│                                        │
│ GPT-Realtime-2 모델                    │
│ ├─ 음성 입력: 1,250,000 tokens · $40.00│
│ ├─ 음성 출력: 1,670,000 tokens · $106.88│
│ └─ 예상 합계: $146.88                  │
│                                        │
│ 단가 기준: 입력 $32/1M, 출력 $64/1M    │
└─────────────────────────────────────────┘
```

### 4️⃣ 대화별 상세 목록 (테이블)

#### 4-1. 테이블 헤더
```
┌─────────────────────────────────────────────────────────────────────┐
│ 대화 제목        │ 시작 시간    │ 소요 시간 │ 메시지 │ 예상 비용   │
├─────────────────────────────────────────────────────────────────────┤
```

#### 4-2. 테이블 행 예시 (5개 표시)
```
│ AI 면접 준비      │ 2026-06-08  │ 12.5분   │ 34개  │ $3.42      │
│ 매 10:30         │             │          │       │            │
├─────────────────────────────────────────────────────────────────────┤
│ 일상 회화        │ 2026-06-07  │ 8.3분    │ 21개  │ $2.18      │
│ 매 14:15         │             │          │       │            │
├─────────────────────────────────────────────────────────────────────┤
│ 비즈니스 이메일  │ 2026-06-06  │ 15.7분   │ 48개  │ $4.05      │
│ 매 09:45         │             │          │       │            │
└─────────────────────────────────────────────────────────────────────┘
[더보기] - 전체 대화 목록으로 이동
```

#### 4-3 테이블 인터랙션
- **클릭**: 해당 대화 상세 정보 모달 열기
- **정렬**: 비용, 시간, 메시지 수로 정렬 가능
- **검색**: 대화 제목으로 필터링

### 5️⃣ 페이지 추가 기능

## 📊 계산 로직

### 사용량 계산
```javascript
// 분당 예상 토큰 사용량
INPUT_TOKENS_PER_MINUTE = 150
OUTPUT_TOKENS_PER_MINUTE = 200

// 토큰 계산
inputTokens = (totalSeconds / 60) * INPUT_TOKENS_PER_MINUTE
outputTokens = (totalSeconds / 60) * OUTPUT_TOKENS_PER_MINUTE

// 비용 계산
AUDIO_INPUT_PRICE = $32/1M tokens
AUDIO_OUTPUT_PRICE = $64/1M tokens

inputCost = (inputTokens / 1_000_000) * AUDIO_INPUT_PRICE
outputCost = (outputTokens / 1_000_000) * AUDIO_OUTPUT_PRICE
totalCost = inputCost + outputCost
```

```

   ↓

   ↓

   ├─ id, title, startedAt
   ├─ durationSeconds
   ├─ messages.length
   └─ coachStyle
   ↓
4. 사용량 계산
   ├─ 총 시간
   ├─ 총 메시지
   ├─ 예상 토큰
   └─ 예상 비용
   ↓
5. UI 렌더링
   ├─ 요약 카드
   ├─ 비용 분석 카드
   └─ 대화별 테이블
```

## 📋 테이블 컬럼 명세

| 컬럼명 | 타입 | 포맷 | 정렬 가능 | 설명 |
|--------|------|------|---------|------|
| 대화 제목 | string | - | ✅ | 대화의 제목 또는 설명 |
| 시작 시간 | timestamp | YYYY-MM-DD HH:mm | ✅ | 대화 시작 시간 |
| 소요 시간 | number | 분 (소수점 1자리) | ✅ | 대화 지속 시간 |
| 메시지 | number | 개 수 | ✅ | 주고받은 메시지 수 |
| 예상 비용 | number | USD ($0.00) | ✅ | 이 대화의 예상 비용 |
